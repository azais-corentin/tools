# Getting Started

Welcome to the **docs** tool. Every markdown file in
`src/routes/docs/content/` is published automatically at a stable URL derived
from its filename.

## How it works

1. Drop a `.md` file into the content folder.
2. Its filename becomes the URL slug (e.g. `getting-started.md` → `/docs/getting-started`).
3. The first `# heading` becomes the document title.

That's it — no configuration required.
